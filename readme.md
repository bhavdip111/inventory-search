### INSTALLATION STEPS && TEST ###

1. Take clone from git repository
	https://github.com/bhavdip111/inventory-search.git

2. Install and update composer 
	~ composer install
	~ composer update

3. Make env file from env.example to set default config setting

4. Generate key 
	~ php artisan key:generate

5. Run migration and seeder
	~ php artisan migrate --seed

6. Run project
	~ php artisan serve

7. Add/Edit product

8. Search product using ajax
